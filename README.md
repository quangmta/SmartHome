# Smart Home
## Data from server via UART: `prefix`+`value`
prefix:
+ _"t"_: Temperature
+ _"v"_: Speed of fan
+ _"f"_: fan control
+ _"h"_: heater control
+ _"c"_: frequency converter control 
+ _"p"_: Kp
+ _"i"_: Ki
+ _"d"_: Kd
+ _"r"_: request
For example: command `t25.5` sets the target temperature to 25.5 degrees
