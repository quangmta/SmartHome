# Smart Home
## Data from server via UART: `prefix`+`value`
prefix:
+ "t": Temperature
+ "v": Speed of fan
+ "f": fan control
+ "h": heater control
+ "c": frequency converter control 
+ "p": Kp
+ "i": Ki
+ "d": Kd
+ "r": request
For example: command `t25.5` sets the target temperature to 25.5 degrees
